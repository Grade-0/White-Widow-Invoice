White Widow Compact Invoice Calculator

Files included:
- index.html
- assets/ww_logo.png
- assets/farmdrop.jpg

How to use on GitHub Pages:
1. Upload the contents of this folder to a GitHub repository.
2. Make sure index.html stays in the root of the repo and the assets folder stays beside it.
3. Enable GitHub Pages for the repository.
4. Open the GitHub Pages link in a browser.

Calculator notes:
- Made for GTA V RP server use.
- Only required field is budtender in-game / Discord name.
- Customer in-game name and in-game phone are optional.
- No taxes are included anywhere.
- Budtender commission is 30% of the invoice total.
- Use the Export PDF button, choose Save as PDF in the browser print window, then post the invoice in Discord for records.
